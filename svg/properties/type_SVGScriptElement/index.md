===Remarks===

If  you do not provide a '''type'''  property, the value of '''contentScriptType''' on the [[svg/elements/svg|'''svg''']] element is used, which  defaults to '''application/ecmascript'''. If a [[svg/elements/script|'''script''']] element falls outside of the outermost '''svg''' element and  you do not provide the '''type''' attribute, the value of '''contentScriptType''' on the '''script''' element  defaults to '''application/ecmascript'''.
