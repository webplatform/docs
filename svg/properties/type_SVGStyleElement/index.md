===Remarks===

The '''type'''   property  specifies the style sheet language of the [[svg/elements/style|'''style''']] element's contents. The style sheet language is specified as a content type (for example, '''"text/css"''').

If  you do not provide a type, the value of '''contentStyleType''' on the [[svg/elements/svg|'''svg''']] element is used, which  defaults to '''"text/css"'''. If a [[svg/elements/style|'''style''']] element falls outside of the outermost '''svg''' element and  you do not provide the type, the value of '''contentStyleType''' on the '''type''' element defaults to '''"text/css"'''.
