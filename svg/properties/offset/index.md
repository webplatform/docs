===Remarks===

The ''offset'' is a number  (typically ranging from 0 through 1) that indicates where the gradient stop is placed. For linear gradients, the offset represents a location along the gradient vector. For radial gradients, the offset represents a percentage distance from the ([[svg/properties/fx|'''fx''']],[[svg/properties/fy|'''fy''']]) point to the edge of the largest  (that is, outermost) circle.
