===Remarks===

If  the ''newItem''  item is already in a list,  the '''insertItemBefore''' method removes it  from its previous list before it is inserted into this list.
|Import_Notes=

===Syntax===

 ISVGTransform retVal = ''object.''insertItemBefore(newItem, index);
