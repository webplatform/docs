## Summary

Reflects the value of the Cascading Style Sheets (CSS) right attribute.

Property of [css/cssom/properties](/css/cssom/properties)[css/cssom/properties](/css/cssom/properties)

## Syntax

``` js
var result = element.pixelRight;
element.pixelRight = value;
```

## Notes

### Remarks

The **pixelRight** property reflects the value of the Cascading Style Sheets (CSS)Â [**right**](/css/properties/right) attribute. Unlike the [**right**](/css/properties/right) property, the **pixelRight** value is an integer, not a string, and is always interpreted in pixels.
