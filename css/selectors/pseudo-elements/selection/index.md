===Remarks===
The '''::selection''' pseudo-element is the UI element fragments pseudo-element. It is used to select any part of the page that the user has highlighted, including text within an editable text field. This pseudo-element can be applied to the [[css/properties/color|'''color''']] and [[css/properties/background-color|'''background-color''']] properties.
